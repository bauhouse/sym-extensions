Version Extension
------------------------------------

Version: 1.0
Author: Stephen Bau (stephen@domain7.com)
Build Date: 11 August 2009
Requirements: Symphony 2.0


[INSTALLATION]

1. Upload the 'version' folder in this archive to your Symphony 'extensions' folder.

2. Enable it by selecting the "Version" Extension, choose Enable from the with-selected menu, then click Apply.

3. You can now view the Version setting in the configuration file (/manifest/config.php) from within the Symphony administration interface.


[USAGE]

- A "Version" field is added to the Symphony Preferences page that provides a means of viewing the useragent setting in the configuration file.
