File Manager
------------------------------------

Version: 0.8.2
Author: Symphony Team (http://symphony21.com)
Build Date: 31st Mar 2008
Requirements: Symphony 2.0.2 or greater

Note, this is a beta extension.


[INSTALLATION]

1. upload this folder, 'filemanager', to your /extensions folder

2. navigate to 'System' > 'Extensions' and enable it

3. Set the root browse location from the 'Preferences' area

4. Use the top level navigation to browse

[KNOWN BUGS]

1. Unable to set/change permissions (http://github.com/pointybeard/filemanager/issues#issue/3)

2. Renaming directory fails (http://github.com/pointybeard/filemanager/issues#issue/4)
