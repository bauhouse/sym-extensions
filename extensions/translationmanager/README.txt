
Translation Manager
------------------------------------

Version: 0.9
Author: Marcin Konicki (http://ahwayakchih.neoni.net)
Build Date: 24 December 2008
Requirements: Symphony version 2.0 or later.


[INSTALLATION]

1. Upload the 'translationmanager' folder in this archive to your Symphony 'extensions' folder.

3. Enable it on System > Extensions page: select the "Translation Manager",
   choose Enable from the "With selected..." menu, then click "Apply" button.

4. Go to System > Translations and have fun :).
